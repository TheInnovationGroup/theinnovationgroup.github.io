About The Innovation Group

The Innovation Group performs in-house software R&D to produce educational materials based on open-source tools, and to provide high-value consulting to solve challenging business and technical problems.

We are comfortable with anything from small MVPs to large-scale datasets and production systems serving millions of people. We produce clean and simple designs along with ultra-fast and reliable system architecture and services, built on known best practices and state-of-the-art technology.

Our Website

Check out the source code of this website here for a look into how we build things.

This website was built entirely from scratch using the latest features available in pure HTML5, CSS3, and JavaScript, with no additional framworks, libraries, or plugins.

Screen metrics are calculated on-the-fly by applying geometric math to the exact current height and width of the screen, resulting in components that fit perfectly in proportion to each other along known-good lines - completely eliminating the usual guesswork and bolt-on junk associated with responsive web design.
